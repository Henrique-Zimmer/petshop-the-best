# PetShop The Best

Sistema web simples para a fase 1 do Projeto da Disciplina.

## Tecnologias utilizadas

- HTML

## Páginas

- `index.html`
- `produtos.html`
- `servicos.html`
- `contato.html`

## Requisitos atendidos

- Cabeçalho com informações do petshop e navegação.
- Visualização de produtos por categoria.
- Dois produtos da categoria acessórios.
- Dois produtos da categoria rações nao perecíveis.
- Dois produtos da categoria higiene e limpeza.
- Serviços de banho e tosa com e sem tele-busca.
- Rodapé com autoria, contato e links.
